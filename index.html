Firebase & React Project

Life List

Using Firebase and React, Life List helps users keep track of their busy lives. After logging in, users are able to create lists and add to them, with other users being able to contribute to them as well. It's a perfect app to help organize the next family potluck or organize a party!

Used: React, Gulp, HTML5, CSS, Sass, Responsive Design, Firebase, Accessible Design.

See Live GitHub

API, Firebase & React Project

Purrrfect Friends

Using the PetFinder API, Firecase and React, Purrfect Friends helps connect users with cats and dogs in their area that are up for adoption, based on their area and perferences. Once logged in with GMail Authentication, users get a completely personlized experience where they can save pets they're interested in to their profile where they're saved for later viewing. Find your next furrrever friend today!

Used: React, Gulp, Firebase, HTML5, CSS, Sass, Responsive Design, API, Accessible Design.
