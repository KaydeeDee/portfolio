Portfolio

Here are the look at the projects I've completed. Feel free to contact me if you have any questions about them!

The Awkward Party Goer

Using the LCBO API and the New York Times "Most Shared" API, The Awkward Party Goer let's users choose the type of party they're going to and their price range and gives them appropriate suggestions for talking points to help break the ice and a drink to pick up at the LCBO.

Used: Javascript, JQuery, HTML5, CSS, Sass, Responsive Design, APIs, Accessible Design.

See Live GitHub

Let's Get Unparliamentary!

From the extremely tame to oddly personal, Let's Get Parliamentary! lets users experience what it's like to be insulted like a Canadian MP with randomally generated insults made up of "banned" parlimentary words and terms.

Used: Javascript, JQuery, HTML5, CSS, Sass, Responsive Design, Accessible Design.

See Live GitHub

Life List

Using Firebase and React, Life List helps users keep track of their busy lives. After logging in, users are able to create lists and add to them, with other users being able to contribute to them as well. It's a perfect app to help organize the next family potluck or organize a party!

Used: React, Gulp, HTML5, CSS, Sass, Responsive Design, Firebase, Accessible Design.

See Live GitHub

Purrrfect Friends

Using the PetFinder API, Firecase and React, Purrfect Friends helps connect users with cats and dogs in their area that are up for adoption, based on their area and perferences. Once logged in with GMail Authentication, users get a completely personlized experience where they can save pets they're interested in to their profile where they're saved for later viewing. Find your next furrever friend today!

Used: React, Gulp, Firebase, HTML5, CSS, Sass, Responsive Design, API, Accessible Design.

See Live GitHub
